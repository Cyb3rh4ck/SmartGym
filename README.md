**SmartGym** is an intelligent workout tracker built for Android using modern development practices. It goes beyond simple logging by allowing users to create custom routines, execute active workout sessions, and receive smart weight recommendations based on past performance.

*   **Smart Tracking:** Log exercises with details on weight, reps, and RPE (Rate of Perceived Exertion).
*   **Routine Management:** Create, edit, and delete custom workout routines (e.g., "Push Day", "Leg Day").
*   **Active Workout Mode:** Interactive checklist mode to guide you through your daily routine step-by-step.
*   **AI Insights:** Basic logic engine that suggests weight increases based on previous RPE and consistency.
*   **History & Analytics:** View your complete workout history with swipe-to-delete functionality.
*   **Modern UI:** Clean, dark/light mode compatible interface built entirely with Material 3.
